6种原型继承方式

六种继承方式

继承方式1 - 默认的原型继承( 很常用 )

继承方式2 - 覆写构造函数的显式原型( 很常用 )

继承方式3 - 给显式原型混入属性( 很常用 )

Eg：
extend(Student.prototype, Person.prototype);
//这里只是把Person.prototype对象里面的东西copy到Student.prototype里面，并没有复写Student.prototype
//Student.prototype没有被复写，只是扩展了里面的内容，Person.Prototype也没有改变

继承方式4 - Object.create( 很少用 )

继承方式5 - 借用Object.create方法覆写显式原型( 很少用 )

继承方式6 - 复合式原型继承( 很少用 )
