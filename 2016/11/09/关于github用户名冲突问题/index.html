关于github用户名冲突问题

关于github用户名冲突问题解决办法：

  • 1、克隆下来github上的一个库，然后git add .==>git commit -m “”==>git push,
    此时会提醒你重新改用户名和密码，按提示进行重新修改
  • 2、控制面板==》所以控制面板==》凭据管理器==》普通凭据，进行删除，然后继续进行push的时候就会提醒重新弹窗让你设置用户名和密码啦，在设置一下就好啦
